# CV Application

A simple React-based CV builder where users can add, edit, and display their personal, educational, and professional information.

---

## 🚀 Live Demo

🔗 **Live Site:** https://cv-application-pink.vercel.app/

---

## ✨ Features

* Add General Information (Name, Email, Phone)
* Add Educational Details
* Add Practical Experience
* Edit and Update submitted data
* Component-based React structure
* Clean and minimal UI

---

## 🛠️ Tech Stack

* React (Vite)
* JavaScript (ES6+)
* CSS

---

## 📁 Project Structure

```
src/

 │    └── PracticalExperience.jsx
 ├── styles/
```
